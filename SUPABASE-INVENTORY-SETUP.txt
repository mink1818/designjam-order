DESIGN SOCKS V6.0.0 Supabase 실시간 재고 설치

1) Supabase 프로젝트 접속
2) 왼쪽 SQL Editor → New query
3) SQL/V6.0.0-SUPABASE-ERP-INVENTORY.sql 파일 전체 복사 → Run
4) Table Editor에서 inventory_items, inventory_movements 생성 확인
5) 프로그램 배포 후 관리자 로그인 → V6 ERP 재고센터
6) 최초 1회 상품 품번 동기화 실행

여러 PC 사용
- 각 PC에서 동일한 배포 주소로 접속합니다.
- 관리자 계정으로 로그인해야 재고센터에 접근할 수 있습니다.
- 같은 계정 동시 로그인도 가능하지만 작업자별 관리자 계정을 권장합니다.
- 모든 PC는 자동으로 동일한 Supabase 프로젝트를 사용하므로 별도 설정이 없습니다.
