Design Jam Order V4.0.2.1 수정사항
- 글자크기 보통/크게/매우 크게가 실제 관리자·거래처 전체 화면에 반영되도록 수정
- 동적 생성 콘텐츠에도 글자크기 자동 적용
- 선택한 글자크기 localStorage 유지
- ERP 품번 검색센터가 상품 데이터 로딩 지연 시 최대 6초간 자동 재연결
- ERP 검색/불러오기 버튼이 연결 완료 후 실행되도록 보완
- 다크모드에서 밝은 배경+밝은 글자 조합 자동 감지 및 대비 보정
- 관리자 주문접수 상세 품번/상품정보 대비 강화
- 거래처 주문내역·장바구니·최근 주문·즐겨찾기 대비 강화
- 바코드 라벨 미리보기는 흰 배경/검정 글자 유지
- 정적 자원 캐시 버전 4024로 갱신
