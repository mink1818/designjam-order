DESIGN SOCKS Windows 백그라운드 스캐너

1. Windows PC에 Python 3.11 이상 설치 시 "Add Python to PATH" 체크
2. 1_설치.bat 실행
3. 2_실행.bat 실행
4. 주문관리 프로그램의 관리자 이메일/비밀번호로 로그인
5. PC 이름, 입고/출고, 수량 선택
6. "백그라운드 스캔 시작" 클릭
7. 크롬·엑셀 등 다른 프로그램이 활성화되어 있어도 빠른 바코드 입력+Enter를 감지

EXE 제작:
- 3_EXE만들기.bat 실행
- dist 폴더의 DESIGN_SOCKS_SCANNER.exe 사용

주의:
- 일반 키보드로 천천히 입력하는 내용은 처리하지 않습니다.
- 바코드 스캐너는 마지막에 Enter(CR)를 보내도록 설정하세요.
- 스캐너 입력은 활성 프로그램에도 보일 수 있습니다. 완전 분리하려면 스캐너 COM/Serial 모드가 필요합니다.
