DESIGN SOCKS 백그라운드 스캐너 V6.2.7

1. 피킹검증
- 관리자 로그인 후 피킹 대기 주문을 한 번 선택합니다.
- 거래처명과 주문번호는 선택한 주문에서 자동으로 적용됩니다.
- 바코드를 스캔하면 해당 주문의 피킹수량이 증가합니다.
- 재고는 스캔할 때 차감하지 않고, 피킹 최종검증 완료 시 한 번 차감됩니다.

2. 입고
- 작업 모드를 입고로 선택하면 바코드마다 ERP 재고가 증가합니다.

3. 수동출고
- 주문과 관계없는 수동 출고에만 사용합니다.
- 이 모드에서만 수동출고 거래처명을 입력합니다.

4. 키보드
- 전체 키보드를 차단하지 않습니다.
- 스캐너가 입력한 바코드는 Enter 시 검색창에서 자동으로 지워집니다.
- 일반 키보드와 Enter는 그대로 사용할 수 있습니다.
- Esc를 누르면 백그라운드 스캔이 중지됩니다.
