DESIGN SOCKS Windows 백그라운드 스캐너 V6.2.4

가장 쉬운 사용법
1. 처음 한 번만 0_처음사용_한번에설치및EXE생성.bat 실행
2. 완료 후 같은 폴더의 DESIGN_SOCKS_SCANNER.exe 실행
3. 이후에는 EXE만 실행하면 됩니다.

스캐너 기능이 수정된 새 버전을 받았을 때
- 새 폴더의 0_처음사용_한번에설치및EXE생성.bat를 한 번 실행하거나, 제공된 새 EXE로 교체합니다.

주의
- 바코드 스캐너는 마지막에 Enter(CR)를 보내도록 설정하세요.
- 일반 키보드로 천천히 입력한 내용은 바코드로 처리하지 않습니다.
