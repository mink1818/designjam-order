디자인 삭스 주문관리 V3.1

추가 기능
1. 관리자 메인 운영 대시보드
2. 오늘 주문, 출고대기, 출고완료, 품절, 승인대기, 전체 거래처 현황
3. 거래처관리: 등급, 할인율, 외상 허용, 관리자 메모, 주문 통계
4. 여러 개 입금계좌 등록 및 기본계좌 선택
5. 공지, 특가이벤트, 신상품, 재입고, 배송안내 게시
6. 거래처 상품화면 공지 표시
7. 관리자 신규주문 / 거래처 출고완료 / 공지 등록 내부 알림 DB 기반
8. 기본 통계 화면

필수 설치
- Supabase SQL Editor에서 V3-1-SETUP.sql 전체 실행
- 이후 전체 프로젝트를 배포

주의
- 기존 테이블과 기능은 삭제하지 않습니다.
- 할인율과 외상 여부는 V3.1에서 관리정보로 저장되며 자동 결제 계산은 후속 단계입니다.
- 브라우저 푸시, 카카오 알림톡, 문자 알림은 후속 연동 단계입니다.
